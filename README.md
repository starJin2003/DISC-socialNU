# SocialNU Frontend - Assignment 5

## Approach
I integrated the React frontend with a local Node.js API. I replaced static mock data with fetch calls to the backend endpoint and managed the API URL using environment variables in a .env file.

## Key Takeaways

- Practiced managing environment variables in a Vite-React environment for security.
